先比数位，数位相同则直接用运算符（<,>,=）比较。

```cpp
bool cmp(string x1,string x2){
	if(x1==x2) return true;//相等返回true
	else if(x1.length()!=x2.length()) return x1.length()>x2.length();
	else return x1>x2;
}
```
